# NOIR Detailing Studio

A premium, conversion-focused website for an automotive detailing studio in Łódź, Poland.

[Live demo](https://eternalstoneinside.github.io/noir-detailing-studio/)

## Overview

NOIR presents detailing services through a fast, polished interface designed around trust and clear customer decisions. The project combines service pricing, portfolio work, testimonials, FAQs, contact routes, and local-business SEO.

## Highlights

- Responsive multi-route React interface
- Motion-rich transitions with reduced-motion support
- Service, pricing, portfolio, testimonial, and FAQ sections
- Local SEO metadata, structured data, sitemap, and robots configuration
- Accessible Radix UI accordion and reusable component utilities
- GitHub Pages-compatible routing and asset paths

## Tech stack

React 19 · TypeScript · Vite · Tailwind CSS · Framer Motion · Radix UI

## Run locally

```bash
git clone https://github.com/eternalstoneinside/noir-detailing-studio.git
cd noir-detailing-studio
npm install
npm run dev
```

Use `npm run build` to create the production bundle and `npm run lint` to run static checks.

## Project structure

```text
src/App.tsx          Main routes and page components
src/index.css        Global styles and Tailwind layers
src/lib/utils.ts     Shared class utilities
public/              Images, SEO files, and favicon
```
